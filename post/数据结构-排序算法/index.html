数据结构中排序是重要内容，其中主要的几大算法有基于比较的冒泡、快排、简单选择、堆排、直接插入、希尔、归并以及不基于比较的计数、基数、桶排

# 数据结构-排序算法

# 基本概念

  1. 稳定排序

    假设在数列中存在a[i]=a[j]，若在排序之前，a[i]在a[j]前面；并且排序之后，a[i]仍然在a[j]前面，则这个排序算法是稳定的

  2. 原地排序

    空间复杂度为 $O(1)$ 的排序算法

# 交换排序

  1. 冒泡排序

    冒泡排序只会操作相邻的两个数据。每次冒泡操作都会对相邻的两个元素进行比较，看是否满足大小关系要求。如果不满足就让它俩互换。一次冒泡会让至少一个元素移动到它应该在的位置，重复 n 次，就完成了 n 个数据的排序工作

    时间复杂度：$O(n^2)$

    空间复杂度：$O(1)$

    稳定性：相邻元素相等时不交换，则为稳定排序

function bubbleSort(nums) { let temp let flag = false // 使用标记位进行优化，没有数据可交换时停止循环 for (let i = 0; i < nums.length; i++) { flag = false for (let j = 0; j < nums.length - 1 - i; j++) { if (nums[j] > nums[j + 1]) { temp = nums[j] nums[j] = nums[j + 1] nums[j + 1] = temp flag = true } } if (flag) { break } } }




2. 快速排序

如果要排序数组中下标 l 到 r 之间的数据，先选择 l 与 r 之间任意一个数据作为分区点 x，遍历 l 到 r 之间的数据，将大于 x 的放右边，小于 x 的放左边，再递归排序两部分，直到区间缩小位 1，则所有数据都完成排序

时间复杂度：$O(nlogn)$

空间复杂度：$O(logn)$，$log_2n$ 次递归

稳定性：不稳定

```javascript
function quickSort(nums, l, r) {
    if (l >= r) {
        return
    }
    let i = l
    let j = r
    let x = nums[i]
    while (i < j) {
        while (i < j && x < nums[j]) {
            j--
        }
        nums[i] = nums[j]
        while (i < j && x > nums[i]) {
            i++
        }
        nums[j] = nums[i]
    }
    nums[i] = x
    quickSort(nums, l, i - 1)
    quickSort(nums, i + 1, r)
}

# 插入排序

  1. 直接插入排序

    将数组中的数据分为两个区间，已排序区间和未排序区间。初始已排序区间只有一个元素，就是数组的第一个元素。插入算法的核心思想是取未排序区间中的元素，在已排序区间中找到合适的插入位置将其插入，并保证已排序区间数据一直有序。重复这个过程，直到未排序区 间中元素为空，算法结束

    时间复杂度：$O(n^2)$

    空间复杂度：$O(1)$

    稳定性：将后面的数据放到相等数据的后面，则为稳定排序

    function insertSort(nums) {
    let temp
    let j
    for (let i = 1; i < nums.length; i++) {
        temp = nums[i]
        j = i - 1
        while (j >= 0 && nums[j] > temp) {
            nums[j + 1] = nums[j]
            j--
        }
        nums[j + 1] = temp
    }
}
